JTechLog Spring Security 3 ACL
==============================

Ez a program a JTechLog (<http://jtechlog.blogspot.com>) blog "Spring Security 3 ACL" cikkéhez készült példaprogram.

`mvnw test` paranccsal futtathatóak a teszt esetek.
